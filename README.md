# SlayTheSpire2_Mods

这是我自用的《杀戮尖塔2》模组集合。

## Mods 简介

### 框架

- **BaseLib**：很多模组共同依赖的基础工具库。
- **ModConfig**：一个模组配置框架，在游戏设置里加了一个"模组配置"标签页，让你可以直接调各个模组的参数。
- **STS-RitsuLib**：一个工具库，ShowPlayerHandCards 需要先装它才能用。
- **UnifiedSavePath**：统一原版与模组版存档路径，使原版与模组版共享进度。

### 功能

#### 懒人
- **SpeedX**：最高 20 倍速！自动跳过动画、自动结束回合、自动捡东西，按 F9 自定义。
- **BetterSpire2**：显示即将受到的伤害、多段攻击标签、长按 R 快速重开、跳过开场动画等一堆贴心功能。
- **BetterModMenu**：增强模组管理菜单，支持配置分组和配置档管理。
- **Rewind**：公平撤回类模组，支持回合倒带、房间检查点回退、公平模式与种子查看。
- **RouteSuggest**：地图上自动画出推荐路线（金色=安全，红色=激进）。

#### 联机
- **DamageMeter**：战斗伤害统计仪表盘，谁在混、谁在 C 一目了然。
- **QuickLink**：联机快速回退工具，房主可回到历史节点，队友可自动回房并自动准备。
- **RemoveMultiplayerPlayerLimit**：将联机人数上限从 4 人提升到 16 人。
- **STS-ShowPlayerHandCards**：联机时实时显示队友手牌，悬停可查看完整卡牌详情。

### 美化

#### 储君

- **RegentCardsAnimeRework**：把 Regent 的卡牌立绘换成可爱动漫风格。
- **Mesugaki**：给 Regent 换上"杂鱼杂鱼♡"风格的臭小鬼语音和外观。
- **BetterSovereignBlade**：可以移动/调整王刃的位置。

## 下载与使用

- 在右侧release下载压缩包，在游戏根目录(一般为Steam/steamapps/common/Slay the Spire 2)新建 mods 文件夹，将解压后的文件放入。
- 使用「[模组管理器](https://github.com/wakaka6/SlaySP2Manger)」（个人推荐），在右侧release下载压缩包后直接拖到「[模组管理器](https://github.com/wakaka6/SlaySP2Manger)」中即可，一步完成。
